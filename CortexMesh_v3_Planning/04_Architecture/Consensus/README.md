# Multi-Agent Consensus Architecture

Status: PROPOSED

---

## Purpose

This workstream defines the architectural consensus model referenced by Policy Section 4.2.

The architecture derives from:

- ARCHITECTURAL_MISSION.md
- AI004 Core Architectural Principles
- AI005 System Model
- AI006 Decision Model
- AI007 Agent Interaction Model

These documents establish architectural constraints.

The MAC series specifies how independent reasoning agents cooperate within those constraints.

Implementation is explicitly out of scope.
