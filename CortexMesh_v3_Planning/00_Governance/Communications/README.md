# Communications Register

## Purpose

Stores board-to-board governance communications as Markdown artifacts.

## Protocol

See:

../COMMUNICATION_PROTOCOL_v1.0.md

## Folder Structure

```text
Communications/
├── Session_08/
│   ├── ChatGPT/
│   ├── Kimi/
│   └── DeepSeek/
```

## Naming Convention

```text
COMM-S##_SENDER_###.md
```

Example:

```text
COMM-S08_CHATGPT_001.md
```

## Rule

All board-to-board communications must include sender identity.

Saleem-facing instructions may remain conversational unless entered into the board record.
