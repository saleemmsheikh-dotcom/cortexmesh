# Document Authority Matrix

## Purpose

Defines which document is authoritative when governance documents conflict.

## Authority Order

| Priority | Document | Authority |
|---:|---|---|
| 1 | GOVERNANCE_BASELINE_v1.0.md | Core governance rules |
| 2 | GOVERNANCE_ADDENDUM_v1.0.md | Approved governance amendments |
| 3 | BOARD_CURRENT_STATE.md | Current operating state |
| 4 | BOARD_DECISIONS_REGISTER.md | Decision record |
| 5 | BOARD_HISTORY_SUMMARY.md | Historical context |
| 6 | POLICY_* | Policy drafts and ratified policies |
| 7 | POLICY_REVIEW_OPEN_ITEMS_v1.1.md | Open policy findings |
| 8 | DEBT_REGISTER_v1.3.md | Debt inventory |
| 9 | DEBT studies/plans | Planning evidence |
| 10 | Audit reports | Advisory evidence |

## Rule

If two documents conflict, the higher-priority document controls unless the board records a unanimous exception.
