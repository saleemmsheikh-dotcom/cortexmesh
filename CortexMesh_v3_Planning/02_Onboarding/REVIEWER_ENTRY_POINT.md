## Governance Rule

Session review packets are historical artifacts.

Do not overwrite previous session packets.

When a new session begins:

1. Create `Session_XX/REVIEW_PACKET.md`.
2. Create `Session_XX/MANIFEST.md`.
3. Update only the current-session pointer in this file.
