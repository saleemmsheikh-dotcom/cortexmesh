# CortexMesh Charter v1.1

## Status
APPROVED

## Mission
CortexMesh is governed as a multi-agent reasoning system with explicit contracts, auditable governance, bounded evolution, and transparent memory/accountability mechanisms.

## Certified Principles
- Specialist routing remains a core design principle.
- Governance violations must be detectable and recoverable.
- External capabilities require process/container isolation.
- In-process guardrails prevent accidents but are not security boundaries.
- Scoring must be deterministic and based on validated evidence.
- Entropy remains available for orchestration monitoring, not authority scoring.

## Baseline Commitment
The v2 baseline is frozen as the certified reference for future v3 planning and architecture debt review.
